# Random Data 
Focado em fornecer dados válidos aleatórios para uso em ambiente de homologação/Testes.

# Como usar
### Gerador de CPF
```
<?php

require_once __DIR__.'/vendor/autoload.php';

use Random\Generate\CPF\CPF;

$cpf    = new CPF();
$result = $cpf->generate();
```
